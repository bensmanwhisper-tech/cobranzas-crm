# Reglas de Proyecto: Cobransas-XD

## Reporte a Obsidian
Cada vez que se discuta, modifique o se resuelva un problema en un área específica del CRM, se debe emitir un reporte.
Ese reporte se debe guardar o anexar como un archivo Markdown en la carpeta de Obsidian del usuario, específicamente en:
`c:\Users\Bensman\Documents\Obsidian ARGUS\Mi cerebro\Cobransas-XD\`

El reporte debe tener el formato:
- Área del CRM afectada.
- Fecha.
- Detalles de la solicitud/problema reportado.
- Cambios realizados o solución aplicada.

**Instrucción:**
Si el archivo correspondiente a la sección (ej. `001 - Clientes.md`) no existe en el directorio de Obsidian, debes crearlo. Si ya existe, añade el nuevo reporte al final de este para mantener el historial. Esto garantizará que quede un registro de lo que le pasó con anterioridad a cada sección.
